自媒体作者日常面临信息过载、热点捕捉滞后、选材决策低效、内容生产标准化程度低等问题，导致发文效率低、爆款率不稳定。该系统旨在打造从 热点实时侦察 → 智能选材决策 → 结构化编辑生产 → 多平台发布的全链路自动化工作流。
#内容工厂 - OpenClaw 爆款文章智能生成器

基于 OpenClaw 的多Agent内容生产工作流，专注于国际时政与热点素材的自动化选材、编辑与发布。

##项目目标
- 实现热点素材的实时侦察与智能筛选
- 自动化生成高质量文章（标题 + 正文 + 配图建议）
- 支持多平台发布流程

## 主要组件
- **Scout Agent**：热点扫描与选材报告生成
- **Editor Agent**：内容结构化编辑
- **Visual Agent**：智能配图建议
- **Publisher Agent**：发布辅助

## 当前功能
- 每日定时扫描多个平台热榜和官方媒体
- 生成结构化选材报告
- 多Agent协作完成内容生产流程

## 技术栈
- OpenClaw（主框架）
- Python 3.11+
- 支持多种大模型（DeepSeek、MiMo 等）

## 运行方式
```bash
openclaw run scout-agent
